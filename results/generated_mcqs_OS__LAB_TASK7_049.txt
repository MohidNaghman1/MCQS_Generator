## MCQ 4

B) By using the `\r` escape sequence.

## MCQ 5
Question: How is output redirected to a file named output.txt using the `echo` command?
A) `echo "text" > output.txt`
B) `echo "text" >> output.txt`
C) `echo "text" | output.txt`
D) `echo "text" < output.txt`
Correct Answer: A
